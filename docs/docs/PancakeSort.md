# パンケーキソート

パンケーキソート(Pancake sorting)は、ある数列を大きさ順に並べる際に先頭から何番目かまでをひっくり返す最小の手数を求める問題である。  
先頭から途中までを一気にひっくり返さなくてはならないのがポイント。  

Microsoftの創始者であるビル・ゲイツが書いた「パンケーキ論文」という学術論文で発表された。
