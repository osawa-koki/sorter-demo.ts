# カクテルソート

カクテルソート(シェーカーソート)は、ソートのアルゴリズムの一つ。  
バブルソートを、効率がよくなるように改良したもの。  

バブルソートではスキャンを一方向にしか行わないのに対し、シェーカーソートでは交互に二方向に行う。  
バブルソートと同じく安定な内部ソートで、最悪の場合の時間計算量はO(n2)である。  

![シェーカーソート](../img/CocktailSort.gif)  

## 計算量

```text
O(n2)
```

## 特徴

- アルゴリズムが複雑。
- パフォーマンスが不安定。
