# バケットソート

バケットソートはあらかじめデータがとりうる値すべての容器(バケット)を順番どおりに並べて用意しておき、値を対応する容器に移すことでソートを行う整列アルゴリズムです。  

![バケットソート](./img/BucketSort.png)  

## 計算量

```text
O(m + n)
# m: バケットの数
# n: バケットが含む要素の数
```

## 特徴

- アルゴリズムが珍しい。
- パフォーマンスが不安定。
